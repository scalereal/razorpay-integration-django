## Getting started:
### Clone this repository:
```
git clone https://github.com/Nishant127/razorpay-integration-django.git
```
### Install pipenv
```
pip install pipenv
```
### Activate environment:
```
pipenv shell
```
### Install dependencies:
```
pipenv install
```
### Start the server:
``` 
python manage.py runserver
```
